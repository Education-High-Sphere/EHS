# Education High Sphere 📘✨

**Education High Sphere** é uma plataforma de educação corporativa desenvolvida com o objetivo de transformar o aprendizado dentro de empresas, oferecendo recursos modernos, intuitivos e personalizáveis para capacitar colaboradores de forma eficaz.

## 🚀 Objetivo

Capacitar profissionais através de conteúdos estratégicos, trilhas de aprendizado personalizadas e acompanhamento de desempenho, promovendo crescimento individual e resultados organizacionais.

## 🧠 Funcionalidades

- 📚 Trilhas de aprendizagem personalizadas por setor ou cargo  
- 📈 Acompanhamento de progresso por gestor e colaborador  
- 🧑‍🏫 Sistema de avaliações e certificados  
- 🔐 Controle de acesso com diferentes níveis de permissão  
- 💬 Fórum interno para discussões e trocas de conhecimento  
- 🗂️ Upload e gerenciamento de materiais internos  

## 👨‍💻 Tecnologias Utilizadas

- HTML, CSS, JavaScript  
- PHP + MySQL (backend e banco de dados)  
- [Framework futuro: React / Laravel / etc., se aplicável]

## 📌 Status do Projeto

🔧 Em desenvolvimento — funcionalidades principais em fase de testes e melhorias.

## 📎 Como rodar localmente

1. Clone este repositório:  
   `git clone https://github.com/Education-High_Sphere/EHS.git`

3. Instale os requisitos (PHP, MySQL etc.)  
4. Crie o banco de dados com base no arquivo `ehsync.sql`  
5. Execute o projeto com um servidor local (XAMPP, WAMP ou outro)
